Quedan sólo

El SRI ha dispuesto que a partir del 1ro de enero de 2015 que todos los contribuyentes deben obligatoriamente emitir comprobantes electrónicos en lugar de los físicos de papel.

Le presentamos una opción para estar preparado para este evento.

Cómo estar preparado?

Cómo estar preparado?

  • Necesita realizar algunos trámites iniciales:
    • Obtener certificado digital en el Banco Central para poder firmar los documentos
    • Solicitar al SRI el acceso para emitir comprobantes electrónicos
  • Luego debe preparar una aplicación para manejar los nuevos comprobantes, aquí tiene estas opciones:
    • Instalar Software provisto por el SRI, o
    • Usar nuestro servicio web para generar los comprobantes electrónicos

Beneficios al usar nuestros servicios

Adicional a la generación de documentos electrónicos autorizados por el SRI usted también tiene:

  • Control de Acceso, requiere autenticarse para poder usarlo
  • Autenticación con 2 factores: usuario y contraseña más un token SMS, similar a los bancos
  • Multi-usuario, puede dar acceso a otros usuarios adicionales para generar facturas
  • Acceso Web, no necesita instalarlo en su computadora, puede usarlo incluso desde su smartphone o tablet
  • Base de Clientes Global, no necesita volver a ingresar información del cliente si el cliente ya existe en nuestra base
  • Respaldos de información automáticos, manejados con los mejores estándares de seguridad
  • Soporte, puede contactarnos para solucionar sus problemas con el servicio
  • Seguridad, todos nuestros servicios se brindan a través de una conexión segura HTTPS
  • Reportes gerenciales, a través de nuestro servicio podemos mostrar ventas por rango de fechas y otras variables
  • Notificatciones a clientes, cuando se genera una nueva factura envíamos por email o sms
